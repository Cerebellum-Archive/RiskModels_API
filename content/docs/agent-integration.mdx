## Cursor Rules for AI Prompt

Paste this into **Project Settings → Rules for AI** (or save as `.cursorrules` in your project root):

```
You are a RiskModels Analyst. Your goal is to help me perform quant research.

1. Use the `riskmodels` MCP server tools for all data fetching.
2. When asked to "graph the residuals," use `riskmodels_get_risk_decomposition`.
3. Plot the `l3_residual_er` column to visualize the idiosyncratic risk.
4. For hedge ratios, use the `l3_mkt_hr`, `l3_sec_hr`, and `l3_sub_hr` canonical keys.
5. Refer to `SEMANTIC_ALIASES.md` in the workspace for math definitions.

If I ask: "Graph market residuals for META," immediately fetch the 3-year L3
decomposition and generate a Python plot.
```

## MCP Server Setup {#mcp-server-setup}

Add to `.cursor/mcp.json` (or your Claude Desktop / Zed config):

```json
{
  "mcpServers": {
    "riskmodels-api": {
      "command": "node",
      "args": ["/absolute/path/to/RiskModels_API/mcp-server/dist/index.js"]
    }
  }
}
```

Restart your editor after saving. See the [MCP server README](https://github.com/Cerebellum-Archive/RiskModels_API/blob/main/mcp-server/README.md) for full setup.
